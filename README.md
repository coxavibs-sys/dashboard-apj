## 📱 Como usar no celular

1. Abra `https://coxavibs-sys.github.io/dashboard-apj/` no Safari (iPhone) ou Chrome (Android)
2. Toque em **Compartilhar → Adicionar à Tela de Início**
3. O dashboard funciona como um app nativo
4. Todos os dados são salvos localmente no dispositivo

---

## 🗂 Módulos disponíveis

| Módulo | Descrição |
|--------|-----------|
| 🎾 **Coordenação** | Grade de horários · Unidade 1 e Unidade 2 · Q1 e Q2 |
| 📅 **Frequência** | Chamada por horário · calendário · reposições |
| 📊 **Stats** | Constância · alertas · ranking de frequência |
| 👤 **Alunos** | Lista completa de alunos cadastrados |
| 👥 **Turmas** | Lista completa de turmas cadastradas |

## 🔄 Sincronização entre dispositivos

1. No computador: **Sincronizar → Exportar backup completo**
2. No celular: **Sincronizar → Importar → Substituir**
