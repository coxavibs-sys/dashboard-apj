## O que é

Dashboard completo para gestão de aulas de tênis, desenvolvido com o **Método APJ** (Aprende · Pratica · Joga).

### Módulos disponíveis

| Módulo | Descrição |
|--------|-----------|
| 🏠 Visão Geral | Status dos módulos, KPIs, roadmap |
| 🏋️ D1 Exercícios | Banco de exercícios APJ com filtros |
| 🎾 D2 Coordenação | Grade de quadras + planejador de aula |
| 📅 D3 Frequência | Chamada do dia, reposições pendentes |
| 📊 D5 Stats | Avaliação APJ, radar 5 dimensões |

## Como usar no celular

1. Abra o link acima no **Safari** (iPhone) ou **Chrome** (Android)
2. Toque em **Compartilhar → Adicionar à Tela de Início**
3. O dashboard funciona como um app nativo
4. Todos os dados são salvos localmente no dispositivo

---

## Backup dos dados

Os dados ficam no `localStorage` do browser. Para não perder nada:

1. Abra o dashboard → toque em **⇅ Sincronizar**
2. Clique em **⬇ Fazer Backup** → salva um arquivo `.json`
3. Guarde o arquivo no Google Drive ou iCloud

## Tecnologia

- HTML + CSS + JavaScript puro (sem dependências externas obrigatórias)
- Arquivo único `index.html` — sem build, sem servidor
- Funciona offline após o primeiro acesso
